# simple-helloworld

### Features:
- Simple print statements
- Input and output
- Basic math operators

### Requirements:
- Ruby 2.3.1+ (https://www.ruby-lang.org/en/documentation/installation/)

### Sample output:
```
ruby simple-helloworld.rb
```
```
HelloWorld!

What is your name?
Adith
Your name is Adith.

a = 10
b = 7
a + b = 17
a - b = 3
a * b = 70
a / b = 1.4285714285714286
a ^ b = 10000000
```
